# Duplicate Review Decision

Every incoming original will be compared against existing archive media.

## Rules

- Possible duplicates are never automatically rejected.
- Possible duplicates enter manual Admin review.
- Normal users are not burdened with duplicate decisions.
- Admin may add as new, reject, link as alternate or merge metadata.
- Rejected duplicate audit records remain permanently.
- Quarantined rejected files may be removed oldest-first when storage is tight.
- Approved originals are never included in automatic storage cleanup.
