# Original Preservation Decision

Original files are sacred.

## Rules

- Originals are never overwritten.
- Originals are never used for normal website browsing.
- Originals receive checksums.
- Original filenames and technical metadata are retained.
- Better later scans may become preferred originals.
- Earlier originals remain linked as alternate source versions.
- Approved originals are never automatically removed.
- Derived versions may be rebuilt.
