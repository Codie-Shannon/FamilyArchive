# Decision: exact hash equality creates a candidate only

A SHA-256 match proves integrity equality, not preferred provenance, metadata ownership, custodianship or the correct lifecycle decision. Group 06 therefore records an explicit `pending_review` candidate with confidence `1.0000` and never sets `confirmed_duplicate` automatically.

Retained quarantine objects, approved originals and derivative inventories remain unchanged. Manual decisions are deferred to a later approved group.
