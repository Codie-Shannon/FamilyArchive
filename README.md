# Family Archive

Family Archive is an archive-grade private family media preservation platform.

The system is designed to preserve original family media, create optimized
viewing versions, detect possible duplicates, manage human-reviewed metadata,
and provide safe family access.

## Group 1

Group 1 establishes:

- Laravel and Livewire application foundation
- MySQL database connection
- Owner authentication
- Owner-only administration area
- archive architecture documentation
- automated access tests

No real family media is imported during Group 1.

## Core Rule

Originals are sacred. They are never overwritten or automatically deleted.
